【功能说明】
  腾讯视频通话 开发SDK。
  1. MP4本地文件解析
  2. 支持实时音视频房间，推流，拉流
  3. 本地录制

【主要接口模块】

   +---------------------------------------------------------------------------+
   |                                                                           |
   |                             your logic code                               |
   |                                                                           |
   |                                                                           |
   +---------------------------------------------------------------------------+
                ^                     |       ^                    |
                |                     |       |                    |
                |                     v       |                    v
   +-------------------------+   +-----------------+   +-----------------------+
   |                         |   |                 |   |                       |
   |                         |   |                 |   |                       |
   |  ITRTCMediaStreamReader |   |   ITRTCCloud    |   |    ITRTCMediaRecorder |
   |                         |   |                 |   |                       |
   |                         |   |                 |   |                       |
   +-------------------------+   +-----------------+   +-----------------------+

  箭头表示 音频/视频数据的流向。
  ITRTCMediaStreamReader --> ITRTCCloud 可以实现推MP4文件到实时音视频房间，满足教育场景下的推课
                                        需求。
  ITRTCCloud --> ITRTCMediaRecorder     将实时音视频房间回调的音视频数据，录制成文件，满足服务端
                                        本地录制需求。
  组件说明
  ITRTCMediaStreamReader     MP4解析组件，设置一个MP4本地文件路径，回调音频和视频帧数据，
                             头文件 TRTCMediaStreamReader.h， TRTCMediaStreamReaderDef.h
  ITRTCCloud                 实时音视频通话组件，和ios，android，windows，Mac，webrtc互通，
                             主要接口函数，是进房，退房，发送音视频数据（推流），回调房间其他主播
                             音视频数据（拉流），事件通知。
                             头文件 TRTCCloud.h，TRTCCloudCallback.h，TRTCCloudDef.h，
                             TXLiteAVCode.h
  ITRTCMediaRecorder         音视频录制组件，ITRTCCloud回调的音视频数据，送到该组件，会录制成本地
                             Flv文件，仅支持单流录制，支持纯音频，纯视频，音视频三种类型的文件输出。
                             头文件 ITRTCMediaRecorder.h
  对接说明
  Your logic code            您可以根据您的业务需求组合使用以上组件。可以参考 main.cpp
  

【开发语言】
  c++

【运行平台】
  支持 Centos 7.6+ x86_64 和 Ubuntu 18.04+ LTS x86_64 系统

【MP4推流测试】
  基于腾讯视频通话 提供的测试账号，30分钟搞定推流测试。


1. cmake 工具安装 版本要求 2.8+ （耗时预计 5分钟）
  Centos 系统下使用如下命令
   +----------------------------------------------------------------------------+
   |                                                                            |
   |   > yum -y install cmake                                                   |
   |                                                                            |
   +----------------------------------------------------------------------------+
  Ubuntu 参考下面命令进行安装
   +----------------------------------------------------------------------------+
   |                                                                            |
   |   > sudo apt-get install cmake                                             |
   |                                                                            |
   +----------------------------------------------------------------------------+

2. 依赖库安装 z压缩库 （耗时预计 3分钟）
  Centos 系统下使用命令行如下
   +----------------------------------------------------------------------------+
   |                                                                            |
   |   > yum  -y install zlib-devel                                             |
   |                                                                            |
   +----------------------------------------------------------------------------+

  Ubuntu 参考下面的安装命令
   +----------------------------------------------------------------------------+
   |                                                                            |
   |   > sudo apt-get install zlib1g.dev                                        |
   |                                                                            |
   +----------------------------------------------------------------------------+
3. 解压并编译示例程序 （耗时预计 1分钟）
   +----------------------------------------------------------------------------+
   |                                                                            |
   |   > cd examples/pushtotrtc                                                 |
   |   > cmake .                                                                |
   |   > make                                                                   |
   |                                                                            |
   +----------------------------------------------------------------------------+  
  然后在 examples/pushtotrtc  目录下生成程序 demo。

  将申请到的accounts.txt 测试账号信息文件 放在 examples/pushtotrtc 目录下。

  运行 
   +----------------------------------------------------------------------------+
   |                                                                            |
   |   > ./demo 66666 ../resource/ruguo-640x360.mp4                             |
   |                                                                            |
   +----------------------------------------------------------------------------+
  66666 是实时音视频房间号，../resource/ruguo-640x360.mp4要推流的mp4文件。

4. “腾讯视频通话”程序 手机端安装 （耗时预计 5分钟）
  方式1. 在PC端用浏览器打开网址 https://cloud.tencent.com/document/product/647/17021  找到移动端demo 手机扫二维码安装。

  方式2. android 手机打开“腾讯应用宝”， 搜索“腾讯视频通话”第一个深色背景的腾讯云Logo的应用，选择安装。
5. 手机运行“腾讯视频通话”程序
  “请输入房间号：” 请填写 66666；
  “请输入用户名：” 用默认值即可,如 123456；
  “视频源” 用默认选项 “摄像头”；
  点击“创建并自动加入该房间”，开始进入 房间号为 66666 的房间。

  一切ok的话 手机端应该能看到mp4的声音和画面。

【userSig签发SDK集成】
  详见：https://github.com/tencentyun/tls-sig-api.git 下面的README.md，按照指引安装集成。
  两点说明：
  1. 编译后会生成一个工具 tls_licence_tools 可以用于验证生成的userSig是否ok，若出错会有提示信息。
  2. 提供了2个版本的接口
     I. v1版本基于公私钥数字签名机制，对应的接口gen_sig()/tls_gen_signature_ex2_with_expire()
     II. v2版本基于 HMAC机制（基于hash的消息验证码机制，hash函数 sha-256），对应接口 gen_sig_v2()
  II 相对于 I，性能方面有很大提升，确保统一服务端签发，同时确保私钥不泄露，安全性方面也是OK的。
